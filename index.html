About The Project

This Optimized Distance Calculator helps to calculate the optimal travel route by road between Indian cities. Dijkstra's single source shortest path algorithm is being used to calculate the optimal route between two cities.

This project was built to demonstrate the use of Data Structures in real life applications. Different Indian cities are cosidered as nodes and path between them are considered as edges. These nodes connect with each other through various edges to form a graph. A doubly linked list is used as to store the recently searched results.

Tech Stacks Used: HTML, CSS, Bootstrap, Javascript, Leaflet.JS
